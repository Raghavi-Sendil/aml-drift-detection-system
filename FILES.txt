USAGE EXAMPLES
==============

1. QUICK START
   python quick_start.py
   - Interactive menu
   - Generate sample data
   - Run full workflow

2. WEB INTERFACE
   streamlit run web_interface.py
   - Upload data via browser
   - Visual dashboards
   - Interactive classification

3. COMMAND LINE
   python cli.py train --data transactions.csv --output model.pkl
   python cli.py classify --model model.pkl --alert alert.json --explain
   python cli.py detect-drift --model model.pkl --data new_data.csv

4. PYTHON API
   from main import AMLDriftDetectionSystem
   system = AMLDriftDetectionSystem()
   system.upload_and_process_datasets('transactions.csv')
   # ... continue with workflow

5. FULL WORKFLOW EXAMPLE
   python example_workflow.py
   - Complete demonstration
   - Synthetic data generation
   - Training, drift detection, classification

RESEARCH PAPER IMPLEMENTATION
==============================

The system implements all methodologies from:
"AML Alert Classification Under Transaction Behavior Shifts for False Positive Reduction"
by Raghavi Sendil, Sai Santhosh Channaka, and Sahil Anjum (Pace University, 2024)

Implemented features:
✅ Feature engineering (exact features from paper)
✅ Temporal window splitting (40%/30%/30%)
✅ Drift detection (PSI, KS, KL)
✅ Drift simulation (+30% size, +20% freq, +50% wire)
✅ Retraining strategy
✅ Performance evaluation across time windows
